Über

Das Modul Schbas erlaubt es, Schulbücher auszuleihen und diese Ausleihen zu verarbeiten.

Begriffsklärung

  • Bücher haben eine ISBN.
  • Exemplare / Buchexemplare sind jeweils eine einzelne Kopie des Buches.
  • Schbas-Vorbereitungsjahr Das Schuljahr, wofür die Ausleihe vorbereitet wird. Wenn zB die auszuleihenden Bücher angezeigt werden, dann nur die für das Schbas-Vorbereitungsjahr.
  • Ausleihzuweisungen sind Zuweisungen von Büchern an Benutzern. Eine Ausleihzuweisung gibt an, dass ein bestimmter Benutzer ein bestimmtes Buch ausleihen soll.

Ablauf

Normal

Tip

Der Ablauf beginnt vor Schuljahreswechsel. Je später damit begonnen wird, desto besser, da die Schülerdaten erfahrungsgemäß aktueller sind und die Ausleihzuweisungen nicht so oft nachkorrigiert werden müssen.

1. Schüler & Daten aktualisieren

Damit das System einschätzen kann, welche Schüler welche Bücher ausleihen sollen, müssen erstmal die Schülerdaten für das nächste Schuljahr eingegeben werden. Dazu erstellt man einen neuen Schuljahreseintrag für das nächste Schuljahr, falls dieser noch nicht existiert. Dann werden die Benutzerdaten für dieses neue Schuljahr importiert. Am Ende setzt man das Schbas-Vorbereitungsjahr auf dieses neue Schuljahr.

2. Ausleihen zuweisen

Nun unter Schbas Dashboard -> Buchzuweisungen automatisch generieren die Ausleihzuweisungen für das neue Schuljahr generieren.

3. Anträge erlauben & erfassen

Damit die Schüler die Schulbuchausleiheformulare ausfüllen können, muss das Rückmeldeformular aktiviert werden (unter Schbas->Dashboard). Zum Ende der Frist zur Formularausfüllung wird dies wieder deaktiviert.

4. Schulbücher ausgeben

Optional

Schüler aktualisieren

Während jedem der oberen Punkte im Ablauf kann der Datensatz der Schüler aktualisiert werden (zB um 5.Klässler mit aufzunehmen). Dabei ist darauf zu achten, dass neue Schüler auch Ausleihzuweisungen zugewiesen bekommen.
